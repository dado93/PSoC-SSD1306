void SSD1306_ClearDisplay ( void  )

Clear current content of the display.

This function clears the current buffer, without sending it to the display. So if the display needs to be cleared, call SSD1306_Display to refresh the display.

uint8_t SSD1306_Display ( void  )

Refresh content of display.

This function sends the updated buffer to the display, so that the new content becomes visible. The whole buffer, which has a dimension dipending on display width and height, is sent over I2C. So make sure to call this function when appropriate.

Return values
#SSD1306_OKif device was found and initialized.
SSD1306_COMM_ERRORif error occurred during I2C communication.

uint8_t SSD1306_InvertDisplay ( uint8_t  invert)

Invert display mode (white-on-black or black-on-white).

Default display mode is white-on-black. The value of the invert parameter allows to choose between white-on-black and black-on-white. If true, black-on-white is enabled.

Parameters
invertinvert mode. Possible values are:
  • #SSD1306_BLACK_ON_WHITE
  • #SSD1306_WHITE_ON_BLACK
Return values
#SSD1306_OKif no error was generated
#SSD1306_COMM_ERRif error occurred during I2C communication
